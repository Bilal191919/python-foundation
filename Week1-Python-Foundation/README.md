# Week 1 Python Foundation

This repository contains Week 1 internship tasks for Python basics, problem solving, and a mini project.

## Folder Structure

```
Week1-Python-Foundation/
|
|-- programs/
|   |-- calculator.py
|   |-- number_tasks.py
|   |-- student_marks_average.py
|   |-- login_system.py
|   |-- list_string_tasks.py
|   |-- pattern_printing.py
|
|-- mini_project/
|   |-- student_management.py
|
|-- screenshots/
|-- README.md
```

## How to Run

Use Python 3.

Examples:

```
python programs/calculator.py
python programs/number_tasks.py
python programs/student_marks_average.py
python programs/login_system.py
python programs/list_string_tasks.py
python programs/pattern_printing.py
python mini_project/student_management.py
```

## Programs Included

- Simple Calculator (add, subtract, multiply, divide)
- Even/Odd Checker
- Factorial Calculator
- Fibonacci Series Generator
- Student Marks Average Calculator
- Simple Login System
- Find largest number in a list
- Reverse a string
- Count vowels in a sentence
- Remove duplicates from a list
- Basic pattern printing

## Mini Project

Student Management System (console based):
- Add student details
- View all students
- Search student by ID/name
- Update student record
- Delete student record
